<h1>🍫 Boreal Doces</h1>

<p>
Este projeto foi desenvolvido com base em uma empresa real de doces artesanais.
A ideia foi criar um site simples, bonito e funcional, que apresentasse os produtos e facilitasse o contato com os clientes.
</p>

<hr>

<h2>🚀 Tecnologias utilizadas</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>

<hr>

<h2>⚡ O que o site faz</h2>
<ul>
  <li>Apresenta os produtos de forma visual e organizada</li>
  <li>Possui navegação simples e direta</li>
  <li>Botão de encomenda com redirecionamento para o WhatsApp com mensagem pronta</li>
</ul>

<hr>

<h2>💡 Sobre o projeto</h2>
<p>
Esse foi um dos meus primeiros projetos práticos, onde pude aplicar na prática conceitos de HTML, CSS e JavaScript.
Busquei focar em um layout limpo, boa experiência para o usuário e algo que realmente pudesse ser utilizado por um negócio real.
</p>

<p>
⚠️ O número de contato foi alterado para preservar a privacidade.
</p>
